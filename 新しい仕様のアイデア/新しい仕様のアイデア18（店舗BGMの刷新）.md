# 新しい仕様のアイデア18（店舗BGMの刷新）

## 概要
店舗（サブエリア）ごとのWeb Audio API BGMを全面的に作り直し、本作の舞台である「新宿」の持つ**哀愁（メロウ）**と**喧騒（ノイズ）**を、8bitチップチューンでより深く表現します。
単なる「単音ループ」から、**「3チャネル（矩形波1・2、三角波ベース）＋ノイズ」**による擬似マルチチャネル演奏への進化を目指します。

## 楽曲のデザインコンセプト

| 店舗 | 情緒的テーマ | 音響的特徴 |
|------|------------|-----------|
| **♨️ 銭湯** | 郷愁と安らぎ（昭和の夕暮れ） | デレイ（エコー）を効かせた、ゆったりとしたヨナ抜き音階。 |
| **🍱 弁当屋** | 庶民の喧騒（忙しない日常） | テンポは速いが、どこか事務的で乾いた矩形波のアルペジオ。 |
| **👕 古着屋** | シティポップの残骸（孤高の都会） | スローテンポなジャズ風。デチューンによる「ヨレ」を加えたエモい旋律。 |
| **♻️ リサイクル** | 吹き溜まりのジャンク（ガラクタの溜まり場） | ノイズや不規則なフィルターによる不安定なリズム。得体の知れない不安感。 |
| **🎯 福引所** | 空虚な祝祭（消費の果て） | マニカルで明るすぎるファンファーレ。高音域の金属的な音色。 |

## 技術的アップグレード（Claudeへの指示内容）

1. **シーケンサーの多層化**
    - `melody`（主旋律：矩形波）、`harmony/bass`（伴奏/低音：三角波）、`noise`（リズム：ノイズ）を並行して管理する。
    - 各音符（Note）に `volume`, `waveType`, `vibrato` などのパラメータを持たせる。

2. **エンベロープ（ADSR）の導入**
    - 全ての音に「Attack（立ち上がり）」と「Release（余韻）」を設け、ピコピコ感の中にも楽器らしい温かみや鋭さを出す。

3. **新宿空間エフェクト**
    - **DelayNode**: 銭湯などの広い空間表現。
    - **BiquadFilterNode**: 通りに面した店舗の「壁越しに聞こえてくる音」などの表現。

## Claude Code用実装指示書

```text
@soundManager.js を修正して、各店舗のBGM（8bitチップチューン）を「新宿の哀愁と喧騒」をテーマに刷新してください。
単音のループではなく、擬似的に複数のチャネルが重なっているような、より音楽的な構成を目指します。

1. **サウンドエンジンの拡張**
   - 既存の `playSynthBGM` を拡張し、複数チャネル（Melody, Bass, Noise）を同時に回せるようにしてください。
   - `OscillatorNode` だけでなく、`BiquadFilterNode`（こもり具合）や `GainNode` によるエンベロープ制御（Attack/Release）を厳密に行い、音の質感を向上させてください。
   - 銭湯用の「エコー（DelayNode）」機能を実装してください。

2. **楽曲データ（旋律）の書き換え**
   - 以下のコンセプトで `SHOP_MELODIES` もしくはそれに代わるデータ構造を定義してください。
   - sento: 和風・スロー（三角波ベース ＋ エコー）。
   - bentoya: アップテンポだが単調。新宿の昼の忙しなさ。
   - furugiya: シティポップ風のメロウな和音（三角波ベース ＋ 少しヨレた矩形波メロディ）。
   - recycle: 不気味なベースライン ＋ メタリックなノイズ。
   - fukubiki: 高速・高音。お祭り騒ぎの裏にある虚無。

3. **ノスタルジックな音色の調整**
   - 古着屋や銭湯では、微弱な LFO（音程の揺れ）を加えて、古いカセットテープやレコードのような質感を再現してください。
   - ノイズチャネルを使って、時折「ザラッ」としたレコードノイズのような環境音を混ぜてください。
```
