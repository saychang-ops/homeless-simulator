# 新しい仕様のアイデア09.md

## 1. 施設の新設と営業時間ルール
すべての店舗・施設に営業時間を設定し、時間外は移動を制限します。

- **共通ルール**: 営業時間外に場所移動を試みた場合、画像は表示せず「ただいまの時間は じゅんびちゅう です。あしたの 〇〇じから 〇〇じまで お店が開いています。」といった案内を表示する。
- **商店街福引所 (西口)**: 
  - **内容**: 福引券3枚で1回ガラポン抽選。金銭や食料以外の「豪華景品」が当たる（はずれあり）。
  - **景品例**: リサイクルショップで高く売れる型落ちの家電（ラジカセ等）、または高性能な衣料品。
  - **営業時間**: 11:00～19:00
- **リサイクルショップ (都庁)**: 
  - **内容**: 拾った高級品（ブランドバッグ・時計等）や、福引で当てた家電・空き缶などを売却（換金）できる。
  - **営業時間**: 10:00～20:00
- **銭湯 (公園)**: 
  - **内容**: 500円支払って入浴することで「えいせい」ゲージが大幅に回復（上昇）する。
  - **営業時間**: 10:00～22:00

## 2. 装備・アイテムシステムの修正
- **同時装備の改善**: マフラーと帽子の同時装備を可能にする。
- **新聞紙の仕様修正**: 新聞紙を装備した際に、マフラーが装備解除されないように修正。
- **名称変更**: 「ニット」を「ニットセーター」に名称変更。
- **新規追加**: ボトムス（ズボン等）の装備品ラインナップを追加。
- **アイテムボックス (拠点)**: 
  - 拠点レベル1（だんボールハウス）以上で設置。
  - 装備を脱いで一時保管し、必要に応じて着脱できるようにする（「のざらし」では使用不可）。

## 3. キャラクターとサバイバルの新要素
- **公園のベテランとの取引**:
  - 有益な情報を聞く前に、ベテランから金銭（少額）を要求されるシーケンスを追加。
  - 支払った場合のみ、医療や支援イベントの重要情報を聞き出せる。
- **熱中症 (Heatstroke)**:
  - **発生条件**: 夏季や長時間直射日光を浴び、体感温度が高すぎる状態が続くと発生。
  - **影響**: 「たいりょく」が急激に減少し、放置すると倒れる。
- **「におい」と不衛生ペナルティ**:
  - 「えいせい」ゲージが極端に低いと、周囲に「におい」が漂う状態になる。
  - この状態では、特定の店舗への入店を拒否されたり、「はなす」の成功率が大幅に低下する。
  - また、「つぶやく」コマンドを実行した際、自虐的に「におうかな。。」とつぶやく演出を追加。

## 4. UI・演出の改善
- **画像ピクセルサイズの統一**: 全画像（背景・拠点・店舗）の縦横サイズを固定し、レイアウトが崩れないように統一。
- **ショップUIの修正**: 売店でもちものリストを開いた際に、画像が崩れるバグを修正。
- **演出追加**: 病気（かぜ等）が完治した際に、「病気が なおった！」というお祝いメッセージをターミナルに表示。
- **BGM**: 店舗（売店、リサイクルショップ等）ごとに専用のBGMを導入。

## 5. 新システム：空き缶拾いとリアル天気予報
- **空き缶拾い**:
  - 路上や公園で「しらべる」を行う際、稀に空き缶を拾えるようにする。
  - リサイクルショップで換金可能。微々たる金額だが、確実な収入源となる。
- **リアルタイム天気予報**:
  - OpenWeather API 等のリアルデータを活用し、現在だけでなく「数時間後〜明日の天気予報」を都庁の掲示板や落ちている新聞で確認できるようにする。
  - これにより、雨や気温変化に対する戦略的な拠点強化や装備変更が可能になる。
